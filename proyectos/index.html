Estructura de detalle

Cada proyecto tiene espacio para documentar el proceso completo.

Las vistas de detalle incluyen objetivo, materiales, imagenes o diagramas, codigo fuente, instrucciones de armado, resultados y posibles mejoras.

Esto deja preparada una biblioteca tecnica para cursos, talleres, demostraciones y futuras generaciones de estudiantes.
